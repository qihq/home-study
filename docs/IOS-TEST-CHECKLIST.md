# iPhone/iPad 真机验收清单

## HTTPS、录制和视频

- [ ] iPhone Safari 在 HTTPS 页面可授权摄像头和麦克风。
- [ ] iPad Safari 在竖屏和横屏均可录制，控制按钮不被安全区遮挡。
- [ ] Safari 与“添加到主屏幕”的 PWA 都能连续录制 15 分钟。
- [ ] 家庭 Wi-Fi 和外网各完成一次上传。
- [ ] 反向代理允许至少 25 MB 的普通上传、至少 50 MB 的 `.flvoice` 上传，且上传/响应超时不少于 10 分钟。
- [ ] 录制中断网 30 秒后恢复，已缓存片段自动上传。
- [ ] 上传未完成时关闭页面，重新打开后可继续上传或明确显示无法恢复。
- [ ] NAS 中有可播放的源视频和 H.264/AAC 720p MP4。
- [ ] 在 iPhone/iPad 下载 720p MP4 后，可通过分享菜单存入照片。

## 学习、AI 和 TTS

- [ ] 中文/英文正式视频均正确进入周/月统计。
- [ ] 粘贴单词、人工默写、显示答案、正确/错误评分和错词复习流程可完成。
- [ ] AI 设置可保存并完成连接测试；页面不回显 API Key 明文。
- [ ] TTS 设置可保存；MiMo 或所选 OpenAI 兼容服务能生成并播放发音。
- [ ] 相同单词在相同 TTS 配置下重复播放正常；更改音色或语速后仍能生成新音频。

## 多人声音和声音包

- [ ] 每位声音使用人均已明确授权；录制前必须勾选授权确认。
- [ ] 使用 HTTPS 的 iPhone/iPad 可录制 8-30 秒的声音样本；样本处理完成后可试听。
- [ ] 为至少两位使用人各创建一个已就绪声音版本，并分别设为默认声音。
- [ ] 在默写或词典朗读中切换到另一位使用人/声音版本后，播放的是所选声音；已创建的默写记录仍保留当时的声音名称快照。
- [ ] 导出 `.flvoice` 后，以正确密码可预览并导入；错误密码不能导入。
- [ ] 导出的声音包不包含 API Key；测试密码已被安全保存，确认遗失后无法恢复。
- [ ] 在“设置 - 数据备份”创建备份；按 `docs/VOICE-PRIVACY-AND-BACKUP.md` 的步骤在非生产副本验证可恢复数据库、参考声音和密钥。
